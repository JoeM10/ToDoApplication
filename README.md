# To-Do App

A simple command-line To-Do List application written in Python.

This project allows a user to view tasks, add new tasks, remove existing tasks, and exit the program through a text-based menu.

## Features

- View all current tasks
- Add a new task
- Remove a task by selecting its task number
- Basic input validation and error handling for user inputs.

## Technologies Used

- Python 3
